## Open Source Projects

### Cloud Native Tools
- [`zookeeper-explorer`](https://github.com/izerui/zookeeper-explorer): Zookeeper 数据浏览器，可视化查看 Zookeeper 节点数据
- [`redis-explorer`](https://github.com/izerui/redis-explorer): Redis 客户端工具，方便浏览和管理 Redis 数据

### AI & Agents
- [`browser-fetch`](https://github.com/izerui/browser-fetch): 专为 AI Agent 和智能体设计的浏览器抓取器，支持转 Markdown、截图
- [`langgraph-vue3-chatbot`](https://github.com/izerui/langgraph-vue3-chatbot): AI Chatbot 前端悬浮对话组件，可集成到业务系统

### Document Processing
- [`pdf-processor`](https://github.com/izerui/pdf-processor): PDF 处理工具，不改变原 PDF 质量的前提下添加 Header 并合并
- [`ocr-micro-service`](https://github.com/izerui/ocr-micro-service): OCR REST API 服务
- [`pp-ocr`](https://github.com/izerui/pp-ocr): OCR 批量识别工具

### Enterprise Tools
- [`work-weixin-spring-boot`](https://github.com/izerui/work-weixin-spring-boot): 企业微信 Spring Boot Starter，支持自建应用和第三方应用
